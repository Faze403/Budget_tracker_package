# budget_tracker

Python 프로그래밍 기말 프로젝트용 가계부 패키지입니다.

## 프로젝트 개요

`budget_tracker`는 일별 수입과 지출을 기록하고, 카테고리별 합계와 월간 리포트를 계산하는 Python 패키지를 만드는 것을 목표로 하고 있습니다.


## 설치 방법

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pip install .
```


## 빠른 시작(quick start)

구현이 완료되면 이 섹션에 바로 실행 가능한 예제를 추가할 예정입니다.

## 주요 기능 설명

예정 기능은 아래와 같습니다.

- 수입 거래 추가
- 지출 거래 추가
- 현재 잔액 계산
- 카테고리별 합계 계산
- 월간 리포트 생성

## 테스트 실행 방법

```powershell
pytest
pycodestyle budget_tracker/
```
